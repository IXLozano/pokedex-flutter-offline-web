# Offline First Pokédex

A cross-platform Flutter Pokédex (Mobile + Web) showcasing:

- Offline-first architecture
- Repository pattern
- Stale-While-Revalidate caching strategy
- Drift-based cross-platform persistence
- Cubit for deterministic state management

## Tech Stack

- Flutter 3.41.1
- flutter_bloc
- Drift
- go_router

---

> This project is built as a technical exercise focusing on architecture, scalability, and offline strategy rather than UI polish.
